Question 01
臨床交班情境題
情境：當你小夜班接班時，發現上一班學姐留下一堆還沒補的醫囑、家屬還在旁邊大喊點滴沒了。
A
【實務戰神模式】「嘖，別吵！事情一件一件來。」迅速開啟高效率模式，按輕重緩急迅速解決。
B
【同理照顧模式】先過去安撫情緒激動的家屬，並貼心地對上一班學姐說：「辛苦了，接下來交給我，你快去下班吧。」
C
【系統優化模式】默默在心裡研究單位的交班流程到底哪裡出漏洞，邊做邊想著應該建立一個防呆機制。
D
【逃離臨床模式】眼神死，內心第 108 次燃起想打開人力銀行看有沒有校護、職護或不用輪班職缺的衝動。
Question 02
應對外部衝突題
情境：病房來了一位 VIP 病患，家屬在護理站頤指氣使，一直質疑你的專業，甚至揚言要找院長投訴。
A
【捍衛專業模式】搬出學理和醫院規定，一字一句專業地「頂回去」，讓對方知道護理專業不可隨便踐踏。
B
【情緒緩衝模式】深呼吸發揮超強同理心：「我知道您很擔心...」先軟化、安撫對方的情緒，再幫他想辦法。
C
【完美防禦模式】默默退回電腦前，把所有護理紀錄、時間點寫得無懈可擊，用完美的證據鏈防範醫療糾紛。
D
【環境撤退模式】內心大喊「夠了！」，好想去一個不需要面對瘋狂家屬、環境與人際關係單純許多的地方。
Question 03
面對工作夥伴題
情境：你今天帶的學妹又出包（漏打針、紀錄寫錯），眼看堆積如山的工作和快要到下班的時間。
A
【閃電全包模式】直接把學妹推開：「算了，我來！」以最快速度自己做完，安全下班最重要。
B
【溫柔帶教模式】把學妹拉到旁邊，溫柔但堅定地說：「別慌，我們一起看看哪裡卡住了。」事後還幫她做心理建設。
C
【除錯檢視模式】開始分析學妹卡關的原因，甚至想動手幫單位做一張「防錯查檢表」，覺得是流程設計不夠好。
D
【強烈共鳴模式】心疼學妹之餘，更加確信這環境真的不能待太久，是時候該規劃換賽道了。
Question 04
能量充電偏好題
情境：辛苦輪班熬了快一個月，終於迎來期待已久的 3 天連休！在連休的第一天，你通常會？
A
【動態釋壓模式】安排滿滿的戶外行程、聚餐、逛街，要把之前因為輪班沒玩到、沒吃到的全部補回來。
B
【情感連結模式】把時間留給家人、閨蜜或另一半，去一家舒服的咖啡廳聊聊天，幫彼此空虛的心靈好好的充電。
C
【自我投資模式】報名了專業的工作坊、研討會，或是關在家裡看書、研究有興趣的新領域，覺得增進自己最踏實。
D
【離線登出模式】關掉所有的鬧鐘和通訊軟體（特別是單位的 LINE 群組！），拉上窗簾躺在床上徹底與世隔絕。
Question 05
五年後理想生活題
情境：閉上眼睛想像一下，如果沒有任何經濟和現實的限制，五年後的你，最希望自己處在什麼樣的工作狀態？
A
【體制核心】成為單位的核心骨幹或阿長，在戰場般的臨床上游刃有餘，享受解決各種疑難雜症的成就感。
B
【深度陪伴】轉任個管師、諮詢師或長照、居家領域，能用一對一、有品質的方式，真正深入去幫助和陪伴需要的人。
C
【跨界影響】在藥廠、醫材大廠、研發部門 or 企業高層，用我的專業知識去優化流程、改善整個醫療或健康環境。
D
【自主自由】擁有一份週休二日、見紅就休的規律工作（如校護、職護、醫美），甚至自己創業做個人品牌。
